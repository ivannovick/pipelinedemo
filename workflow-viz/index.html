Source

Airflow DAG

gdrive_publish_to_stream

Trigger or schedule; orchestrates publish task only.

↓

Python module

gdrive_stream_publisher

Read bytes, JSON envelope (pdf_base64, name, URL).

  • No Postgres
  • No text extraction

↓

Broker

RabbitMQ Stream

Durable log; port 5552; stream pdf.jobs (configurable).

Consume — batch read stream, write Bronze + files

Airflow DAG

pdf_stream_batch_consumer

Scheduled batch poll; subscriber name + store_offset.

↓

Python module

pdf_stream_processing

Decode → pypdf → sanitize → write file → INSERT.

  • Idempotent on (stream, offset)
  • NUL bytes stripped for Postgres

↓

Sink

PostgreSQL

public.pdf_documents — text, paths, stream metadata.
